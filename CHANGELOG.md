# Changelog

## [1.0.0] - 2024-01-09

### Added
- Initial release
- 5 presets: Mobile, Web, Full-Stack Mobile, Full-Stack Web, Monorepo
- LazorKit SDK integration with passkey auth, gasless transactions, biometric onboarding
- React Native + Expo support via create-expo-app
- Next.js support via create-next-app
- TailwindCSS (web) / NativeWind (mobile) styling
- Zustand and Redux Toolkit state management
- Supabase and Firebase backend options
- shadcn and nativewind-ui component libraries
- npm, pnpm, yarn, bun package manager support
- Git initialization option
- Gradient ASCII banner
